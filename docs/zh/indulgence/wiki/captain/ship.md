# 飞船
!!! quote "《古船图鉴：蒲公英I型》"

    “蒲公英I型被涂成欢快的黄色，外壳像一个饱满的种子荚，专门在轨道空间站和星球定居点之间运送乘客和货物。尽管其速度被专业人士所诟病，但温和的加速度却备受乘客的青睐。”

对船长来说，世界上没有什么能比飞船更重要的了。尽管飞船只是*星港公司*废物利用的产物，使用退役的太空摆渡船拼接而成，但在边境星球已经是妥妥的高科技造物了。也正因如此，只有船长这种来自边缘星球的杰出人士才能驾驭它，对普通定居者来说，飞船只是一块废铁而已。

## 舱室 {#cabin}
*舱室*承担了飞船的具体功能，大致可以分成以下几种类型，
- **预制**：
- **生活**：
- **工艺**：
- **功能**：

| 舱室       | 类型 | 工作栏 | 描述                                 |
| ---------- | ---- | ------ | ------------------------------------ |
| 引擎室     | 预制 | 机械师 | 为飞船提供飞行动力和供电。           |
| 驾驶舱     | 预制 | 飞行员 | 对飞船进行飞行控制和导航。           |
| 底仓       | 预制 | 售货员 | 改造为店铺的舱室，用来贩卖物品。     |
| 餐饮室     | 预制 | 厨师   | 为乘员提供食物和饮料，也是社交场所。 |
| 船员室     | 生活 | --     | 乘员的私人舱室。                     |
| 阅览室     | 生活 | --     | 乘员阅读书籍和资料的舱室。           |
| 医护室     | 生活 | 医生   | 对乘员进行治疗和护理的舱室。         |
| 祈祷室     | 生活 | 传道者 | 乘员满足精神需求的舱室。             |
| 初加工舱   | 工艺 | 工匠   | 进行初级工艺加工的舱室。             |
| 精加工舱   | 工艺 | 工匠   | 进行高级工艺加工的舱室。             |
| 园艺舱     | 工艺 | 园丁   | 培育植物的舱室。                     |
| 拆解舱     | 工艺 | --     | 拆解物品的舱室。                     |
| 储能舱     | 功能 | --     | 存储多余电力的舱室。                 |
| 货舱       | 功能 | --     | 存放货物的舱室。                     |
| 通讯舱     | 功能 | --     | 进行远程通讯的舱室。                 |
| 战斗训练舱 | 功能 | 安保   | 日常战斗训练的舱室。                 |

## 改造 {#modification}

### Switch Active Status
Deactivate a _Cabin_ can stop its function, and at the same time, stop its power input/output.
### Maintenance
- A _Mechanic_ would do the routine automatically, the _Captain_ also can assign the maintenance chore manually
- _Captain_ can decide which _Cabin_ could use `spare parts` in maintenance
